## Usage

Download latest version at https://github.com/fireworkweb/fwd/releases and move file to `/usr/local/bin/fwd`

```
fwd install
fwd start
fwd composer install
fwd artisan migrate:fresh --seed
fwd yarn install
fwd yarn dev
fwd down
```
